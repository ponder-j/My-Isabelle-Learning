# 简单类型λ项详解

## 定义

`t = x | c | λx::τ. t | t t`

这是一个**递归定义**，用 `|` 符号表示"或者"，说明一个λ项 `t` 可以是以下四种形式之一：

## 四种基本形式

### 1. x (变元/变量)

最基本的形式，就是一个变量，比如 `x`, `y`, `z` 等。

**例子**: `x`, `n`, `sum`

### 2. c (常元/常量)

预定义的常量，比如数字、布尔值等。

**例子**: `0`, `1`, `True`, `False`

### 3. λx::τ. t (λ抽象/函数定义)

这是定义一个函数的方式：

- `λ` 表示"这是一个函数"
- `x` 是函数的参数名
- `::τ` 表示参数 `x` 的类型是 `τ`
- `.` 后面的 `t` 是函数体（注意这里的 `t` 又可以是上面四种形式之一）

**例子**:

- `λx::nat. x + 1` (接受一个自然数，返回它加1)
- `λf::nat⇒nat. λx::nat. f(f(x))` (高阶函数，应用函数两次)

### 4. t t (函数应用)

把一个函数应用到参数上，即"调用函数"。左边的 `t` 是函数，右边的 `t` 是参数。

**例子**:

- `(λx::nat. x + 1) 5` → 结果是 `6`
- `f x` (把函数 `f` 应用到参数 `x`)

## 综合例子

让我用一个完整的例子展示这些组合：

```
(λx::nat. x * 2) (3 + 4)
```

分解一下：

- `λx::nat. x * 2` 是一个λ抽象(第3种)
  - 其中 `x * 2` 这个函数体本身包含变元 `x`(第1种)和常元 `2`(第2种)
- `3 + 4` 是参数(包含常元)
- 整体是函数应用(第4种)
- 计算结果: `(3 + 4) * 2 = 14`

## 总结

这个递归定义的强大之处在于：通过这四个简单规则的组合，可以构造出任意复杂的函数和表达式！

为了和对象逻辑（如一阶逻辑）中的全称量词(∀)、蕴涵(⟶)和等式(=)进行区别，元逻辑中称为元全称量词（符号为⋀）、元蕴涵（⟹）和元等式（≡）。元全称量词主要用于表达元逻辑层面的自由变元，元蕴涵表明逻辑公式之间的推理关系，用于表达推理规则、引理/定理等。其中，A`<sub>`1`</sub>`⟹...⟹A`<sub>`n`</sub>`⟹B，可以简写为〚A`<sub>`1`</sub>`,...,A`<sub>`n`</sub>`〛⟹ B，含义是A`<sub>`1`</sub>`和…和A`<sub>`n`</sub>`蕴涵或推理出B。
